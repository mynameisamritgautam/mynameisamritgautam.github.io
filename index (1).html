2.3

Parallel Prefix Sum on GPU — CUDA

CUDA C/C++GPUParallel

A two-person project from UTA's GPU programming course: parallel prefix sum (scan) in CUDA C/C++, built in three stages — a sequential CPU baseline, a naive one-thread-per-element kernel, then a work-efficient Blelloch scan using shared memory and coalesced global reads. Course benchmarks measured the optimized kernel at roughly 40x over the sequential baseline on million-element arrays. Backed by a semester of CUDA lab work: multidimensional grids, shared-memory benchmarking, convolutions, reductions, and scans.

STAGES: CPU → NAIVE KERNEL → BLELLOCH + SHARED MEM TEAM: 2 — GPU PROGRAMMING, UTA
2.4

Floating-Point Adder/Subtractor

VerilogDE10-LiteFPGA

A solo Digital Logic II project: a floating-point add/subtract unit designed in Verilog and validated on a DE10-Lite (Intel MAX 10) FPGA. The datapath does what every real FPU must — unpack sign, exponent, and mantissa fields, align mantissas by exponent difference, add or subtract with sign handling, then normalize and repack the result. Integer math is one lab; making the binary point float is another.

OPERATIONS: FP ADD · FP SUB SOLO — DIGITAL LOGIC II, UTA
2.5

Embedded Ethernet Stack with DHCP Client

Embedded CTM4C123ENC28J60

An IoT networking node built on a TI TM4C123 microcontroller and ENC28J60 Ethernet controller over SPI — no OS, no vendor network stack. Implemented the DHCP client state machine (DISCOVER → OFFER → REQUEST → ACK, plus lease renewal and rebinding), a UART command-line interface for configuration, and EEPROM persistence so settings survive power cycles.

PROTOCOLS: DHCP · UDP · IP · ARP SOURCE: github.com/mynameisamritgautam
